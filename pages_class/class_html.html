What's HTML

HTML (HyperText Markup Language) เป็นภาษาที่ใช้สำหรับสร้างเว็บเพจ มีโครงสร้างภาษาโดยใช้ตัวกำกับ (Markup Tag) เพื่อควบคุมการแสดงผลข้อมูล รูปภาพ และวัตถุอื่นๆ ผ่านทาง Web Browser

HTML 5.0

มาตรฐานชองภาษา HTML มีการจัดโครงสร้างและการแสดงผลของเนื้อหาสำหรับ www มาตรฐานใหม่จะมีคุณลักษณะเด่นที่สำคัญเช่น

  • เล่นวิดีโอ
  • แสดงแผนที่
  • เก็บไฟล์แบบออฟไลน์
  • แสดงกราฟฟิก รูปภาพ
  • การป้อนข้อมูลแบบใหม่ เช่น search, number, range etc.
